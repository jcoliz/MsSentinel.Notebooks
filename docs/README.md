# Documentation

This directory contains end-user documentation for the repository. Documents here are instructive and explanatory, providing finalized guidance on using and contributing to the notebook collection.

## User Documentation

### [Getting Started](getting-started.md)
Comprehensive guide for getting started with Microsoft Sentinel data lake notebooks. Covers:
- Prerequisites and setup
- Running existing notebook examples
- Creating your own notebooks with AI assistance
- Understanding Sentinel tables and the provider API
- Publishing and sharing notebooks
- Troubleshooting common issues

### [Repository Structure](repository-structure.md)
Complete overview of the repository organization. Covers:
- Directory purposes and layout
- Workflow for users and contributors
- Configuration management approach
- Documentation requirements
- Security model and best practices
- Category guidelines

## Work in Progress

The [`wip/`](wip/) directory contains planning documents and architectural discussions that are still being refined. These documents help shape future features and conventions but are not yet finalized.
