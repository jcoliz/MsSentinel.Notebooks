# Microsoft Sentinel data lake notebook examples

This repository contains examples of Jupyter notebooks which can run in the Microsoft Sentinel data lake to accomplish various security scenarios. My goal
is to illustrate the possibilies of Sentinel Data lake for your security
scenarios, and teach you how to build your own for your unique needs.
